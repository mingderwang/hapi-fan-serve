# hapi-fan-serve

test keycloak plugin


## Usage

```bash
$ cd hapi-fan-serve
$ npm install
$ npm update --save
$ npm test
$ npm start
```

## Supporting keycloak

because server/keycloak-hapi (is untracked content), you need to git clone it.
```bash
$ cd server
$ git clone https://github.com/keycloak/keycloak-hapi.git
```
